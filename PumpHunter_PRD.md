# PumpHunter — Торговый бот-ассистент для скальпинга волатильных криптовалют

## Описание проекта

**PumpHunter** — Telegram-бот для автоматического сканирования экстремальных движений на рынке криптовалют и предоставления торговых сигналов с детальной аналитикой. Бот помогает трейдеру находить перспективные точки входа в шорт на выдыхающихся пампах и в лонг на начинающихся движениях.

### Ключевая проблема которую решает проект

Трейдер работал по стратегии шорта сильных пампов (+15-70%+) с усреднением по мартингейлу на плече x10. Win rate был 98%, но редкие катастрофические дни (ликвидации на монетах которые продолжили рост) уничтожали недели прибыли. Итог за 3 месяца: убыточный несмотря на огромное количество выигрышных сделок.

**PumpHunter** решает это через:
- Фильтрацию качества сигналов (объём, OI, funding rate, карта ликвидаций)
- Рекомендации по безопасным уровням входа/выхода/усреднения
- Защиту от катастрофических потерь (дневной лимит, умный стоп-лосс)
- Анализ открытых позиций по запросу с рекомендацией держать/выйти/усреднить

### Целевая аудитория

- Активные скальперы работающие на фьючерсах Bybit
- Трейдеры с опытом работы с плечом и волатильными альткоинами
- Те кто торгует вручную но хотят автоматизировать мониторинг рынка

- **Runtime:** Node.js 18+

## Основные функции

### 1. Автоматическое сканирование рынка

**Что делает:**
- Каждые 3-5 минут сканирует топ-500 монет Bybit по суточному объёму
- Вычисляет изменение цены за скользящее окно 60 минут
- Применяет фильтры качества сигнала
- Отправляет уведомление в Telegram при совпадении условий

**Типы сигналов:**

#### ЛОНГ сигнал 🟢
**Условия:**
- Рост +15-25% за последний час
- Объём текущей свечи в 3-5x выше среднего
- Open Interest растёт (новые деньги заходят)
- Funding rate нейтральный или отрицательный (толпа не перегружена лонгами)
- Суточный объём минимум $30M

**Логика:** Ловим начало реального движения до того как толпа запрыгнет.

#### ШОРТ сигнал 🔴
**Условия:**
- Рост +50% и выше за последний час
- Объём на последних 2-3 свечах падает относительно пиковых
- Open Interest стабилен или падает
- Funding rate сильно положительный (+0.3%+)
- Карта ликвидаций: кластер шортов выше уже собран, кластер лонгов ниже есть
- Суточный объём минимум $30M

**Логика:** Памп выдыхается, импульс гаснет, толпа в лонгах — время шортить откат.

### 3. Система оценки качества сигнала

Каждый сигнал получает рейтинг от ⭐ до ⭐⭐⭐:

**⭐⭐⭐ (ВЫСОКОЕ КАЧЕСТВО)** — все факторы совпали:
- Для шорта: объём падает + funding перегрет + OI снижается + карта ликвидаций благоприятна
- Для лонга: объём растёт + OI растёт + funding нейтральный + нет недавних листингов

**⭐⭐ (СРЕДНЕЕ КАЧЕСТВО)** — 2-3 фактора из 4

**⭐ (НИЗКОЕ КАЧЕСТВО)** — только памп без подтверждающих факторов

### 4. Анализ позиции по запросу

Трейдер может запросить анализ уже открытой позиции:

**Команда:**
```
/analyze XYZUSDT short 0.084 -15%
```

**Ответ бота через Claude API:**

```
📊 АНАЛИЗ ПОЗИЦИИ: XYZUSDT

Твой вход: $0.084 (ШОРТ)
Текущая цена: $0.097
P&L: -15% ($45 убыток при позиции $300)

📈 Текущая картина:
— Памп от старта: +89% (был +74% при входе)
— Объём: всё ещё растёт ⚠️
— Open Interest: +12% за последний час ⚠️  
— Funding rate: +0.62% (толпа ещё в лонгах)
— Свечи: импульс не гаснет, новый хай 5 минут назад

🗺 Карта ликвидаций:
— Шортов выше: $1.8M на $0.104
— Лонгов ниже: $3.4M на $0.088

🤖 РЕКОМЕНДАЦИЯ: ВЫХОД 🚨

Объём и OI продолжают расти — памп не выдохся. 
Это не коррекция а продолжение движения.
Усреднение преждевременно, риск выноса высокий.

⛔️ Если держишь — хард стоп $0.104 (следующий кластер шортов)

Альтернатива: зафиксируй -15%, переоцени на новом сигнале.
```

**Возможные рекомендации:**
- **ВЫХОД** — данные против тебя, фиксируй убыток
- **ДЕРЖИ** — признаки разворота появились, подними стоп
- **УСРЕДНЯЙ** — всё совпало, можно добавить по плану

### 5. Ручной скан монеты

```
/scan BTCUSDT
```

Бот проверяет монету по всем критериям и говорит есть ли сигнал сейчас или нет.

### 6. Защита от катастрофических потерь

**Дневной лимит убытков:**
- При депозите $500 → лимит -$50/день (10%)
- Трейдер может настроить свой лимит через `/settings`
- При достижении лимита — бот блокирует новые сигналы до следующего дня
- Отправляет уведомление: "⛔️ СТОП НА СЕГОДНЯ — достигнут дневной лимит"

**Anti-flood сигналов:**
- Одна монета = максимум 1 сигнал раз в 2 часа
- Исключение: если сигнал поменял тип (был лонг → стал шорт)

### 7. Режимы работы

**24/7 с тихим ночным режимом:**
- Дневной режим (08:00 - 00:00 по UTC): все уведомления со звуком
- Ночной режим (00:00 - 08:00 по UTC): только критичные уведомления без звука
- Трейдер может настроить свои часы через `/settings`

### 8. Статистика и логирование

```
/stats
```

Показывает:
- Сегодняшний P&L
- Количество сигналов за день/неделю
- Win rate по отработанным сигналам (если трейдер отмечал входы)
- Лучшие/худшие монеты по результатам

Все сигналы и действия логируются в SQLite для последующего анализа.

### Модули

**1. Scanner Module**
- Подключение к Bybit WebSocket для real-time данных
- Расчёт изменения цены за скользящее окно 60 минут
- Фильтрация по объёму ($30M+ суточный)
- Обновление топ-500 списка раз в день

**2. Signal Analyzer**
- Проверка условий для ЛОНГ/ШОРТ сигнала
- Расчёт рейтинга качества сигнала (⭐-⭐⭐⭐)
- Расчёт рекомендуемых уровней входа/тейка/стопа
- Интеграция с Coinglass API (опционально) для карты ликвидаций

**3. Position Tracker**
- Хранение открытых позиций трейдера
- Мониторинг изменения цены относительно входа
- Алерты при движении против позиции

**4. Claude Integration**
- Формирование промпта с данными позиции
- Запрос к Claude API для анализа
- Парсинг рекомендации и отправка трейдеру

**5. Risk Manager**
- Отслеживание дневного P&L
- Блокировка сигналов при достижении лимита
- Расчёт размера позиции от депозита

### API endpoints которые используем

**Bybit API:**
- `GET /v5/market/tickers` — цены и объёмы всех пар
- `GET /v5/market/kline` — свечи для расчёта движения
- `GET /v5/market/open-interest` — Open Interest
- `GET /v5/market/funding/history` — Funding rate
- WebSocket `/v5/public/spot` — real-time обновления цен

**Coinglass API (опционально, платно ~$50/мес):**
- Liquidation heatmap data
- Альтернатива: парсить с сайта или использовать бесплатные аналоги

**Claude API:**
- `POST /v1/messages` — анализ позиции с контекстом

### База данных (SQLite схема)

```sql
-- Таблица сигналов
CREATE TABLE signals (
  id INTEGER PRIMARY KEY,
  timestamp INTEGER,
  symbol TEXT,
  type TEXT, -- 'LONG' | 'SHORT'
  price REAL,
  change_1h REAL,
  volume_24h REAL,
  quality_score INTEGER, -- 1-3 stars
  entry_levels TEXT, -- JSON массив
  take_profit REAL,
  stop_loss REAL,
  sent_to_user BOOLEAN
);

-- Таблица позиций трейдера
CREATE TABLE positions (
  id INTEGER PRIMARY KEY,
  symbol TEXT,
  type TEXT,
  entry_price REAL,
  current_price REAL,
  pnl_percent REAL,
  opened_at INTEGER,
  closed_at INTEGER,
  status TEXT -- 'OPEN' | 'CLOSED'
);

-- Логи дневного P&L
CREATE TABLE daily_pnl (
  date TEXT PRIMARY KEY,
  total_pnl REAL,
  trades_count INTEGER,
  wins INTEGER,
  losses INTEGER
);
```

## Финальные замечания

**PumpHunter** — это не holy grail бот который гарантирует прибыль. Это инструмент который:

1. **Экономит время** — не нужно вручную мониторить 500 монет
2. **Фильтрует качество** — показывает только перспективные ситуации
3. **Защищает от катастроф** — дневной лимит и умные стопы
4. **Помогает принимать решения** — анализ Claude как второе мнение

Финальное решение по входу/выходу всегда за трейдером. Бот — ассистент, не автопилот.

Успех стратегии зависит от:
- Дисциплины соблюдать стопы и лимиты
- Правильной интерпретации сигналов (⭐⭐⭐ приоритетнее чем ⭐)
- Адаптации под меняющийся рынок (пороги могут требовать корректировки)
